# 可可英语听写自动填充工具

这个工具可以帮助你自动填充可可英语听写页面上的空白处。

## 使用方法

1.  **打开网页**: 在浏览器中打开你需要进行听写的页面（例如：`https://www.kekenet.com/write/...`）。
2.  **打开开发者工具**:
    *   按下键盘上的 `F12` 键。
    *   或者右键点击页面任意位置，选择“检查” (Inspect)。
3.  **切换到控制台**: 在开发者工具窗口中，点击顶部的 "Console" (控制台) 标签。
4.  **运行代码**:
    *   复制 `auto_fill.js` 文件中的所有代码。
    *   粘贴到浏览器的控制台中。
    *   按下 `Enter` 键运行。

## 原理

该脚本会查找页面右侧（在代码结构中）的 `class="container-flex listen-container"` 元素，遍历其中的单词节点。如果发现某个节点包含输入框（即空白处），它会尝试从该节点中提取隐藏的答案文本，并自动填入输入框中。

## 注意事项

*   该脚本仅供学习和研究使用。
*   如果页面结构发生变化，脚本可能会失效。
*   脚本会尝试触发输入事件，以便网页能识别到填入的内容，但某些复杂的网页可能需要手动点击一下输入框才能生效。
